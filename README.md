## Was ich mache

**Suchmaschinenoptimierung (SEO)** — Technische Audits, Content-Strategie, Ranking-Aufbau. Von der Keyword-Recherche bis zum fertigen Maßnahmenplan.

**Generative Engine Optimization (GEO)** — Sichtbarkeit in ChatGPT, Perplexity und Google AI Overviews. Strukturierte Daten, Entity-Aufbau und zitierfähige Inhalte für die KI-Suche.

**Technisches SEO** — Core Web Vitals, Schema Markup, Crawling-Optimierung, PageSpeed, Indexierung. Ich baue Websites so, dass Suchmaschinen sie verstehen.
